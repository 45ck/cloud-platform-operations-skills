# Sample flow

1. Run `iaas-paas-saas-faas-selector` to narrow the service model.
2. Run `public-private-hybrid-cloud-reviewer` to decide the placement strategy.
3. Run `resource-lifecycle-planner` to define ownership, tagging, retirement, and automation.
4. Run `acl-maintenance-reviewer` and `orphaned-resource-auditor` to reduce drift and exposure.
5. Run `patch-management-checker` and `maintenance-window-planner` before significant platform updates.
6. Run `blue-green-readiness-reviewer` or `update-rollback-planner` for safer service changes.
7. Run `migration-wave-planner` for multi-system or staged cloud/platform moves.
